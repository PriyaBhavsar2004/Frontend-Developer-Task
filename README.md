# Frontend Developer Intern Task

## Project Overview
A complete full-stack web application with authentication and dashboard.

## Features
- JWT-based authentication
- Protected routes
- CRUD operations on tasks
- Secure password hashing with bcrypt
- Modular and scalable project structure

## Tech Stack
Frontend: React + Vite  
Backend: Node.js + Express  
Database: MongoDB  
Auth: JWT  

## Run Instructions

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm run dev
